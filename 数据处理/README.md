# 家教排课助手

一个无需安装的本地网页小程序，用于安排家教课程。

## 使用方式

直接双击打开 `index.html`，或在浏览器中打开它即可。

## 功能

- 周一至周日，每天 8:00–21:00 的课表视图
- 每节课固定为两小时，可选择 8:00 到 19:00 的整点开始时间
- 自动拦截同一天、时间重叠的课程
- 支持线上/线下、学生姓名、科目和备注
- 可设定课程计划结束日（默认 2026 年 8 月 1 日）
- 可将课程标注为缺课，系统会在计划结束日后的首个空闲两小时段自动安排补课
- 可切换查看不同周、删除单节课或清空当周课程
- 数据自动保存在当前浏览器；刷新或关闭页面后仍会保留
